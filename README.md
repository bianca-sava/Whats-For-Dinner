# 🍽️ WhatsForDinner

A web application that suggests recipes based on the ingredients you have in your fridge. 

Built with Spring Boot, PostgreSQL, and React (PWA).

## Running locally

1. Create `src/main/resources/application-local.properties` with your DB credentials
2. Start the database: `docker compose up -d`
3. Run the app from IntelliJ or `./mvnw spring-boot:run`
